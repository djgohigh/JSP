 1. 중고차는 역시 호카! 팀프로젝트('21. 11. 08. ~ 12.)
 
 2. 주제 : JSP 및 HTML 활용, CRUD/MVC패턴 구현
 3. 주요기능
    가. 로그인
        1) 사용자의 로그인 데이터(EMAIL)를 조건으로 관리자, 유저로 구분해 메인페이지 이동
        2) 로그인 세션 유지 → 유저의 구매, 판매시 user DB에 저장된 이름 등의 정보 input tag의 value로 자동입력
    나. 조회, 구매주문, 판매주문
        1) 사용자가 원하는 조건으로 조회하고 '구매하기' 버튼으로 해당 차량을 구매주문
           ※ mysql의 limit를 활용, get전송방식의 ?&name=value 의 parameter의 배열값으로 조회페이지의 페이징구현
        2) 유저 메인화면의 '판매' 기능으로 차량기본정보를 입력하고 판매주문
    다. 관리기능
        1) 관리자 메인에서 주문접수, 판매접수 확인
        2) 차량 전체 DB에 대한 추가, 수정, 삭제를 통한 관리
    라. 구현해본 프론트엔드(웹디자인)
        1) 사진 슬라이드(로그인 메인화면, 유저 메인화면)
        2) 팝업기능(관리자 전체차량조회)
 4. 부족했던 점
    가. 페이지 레이아웃에 대한 개념이해, 트렌드에 부합하지 않는 웹디자인 → 부트스트랩 등 활용필요
    나. 차량조회 시 제한된 조건구현 → 가격, 연식, 주행거리 등 범위값(시간부족)
    다. 네트워크 보안 : 'https', 서버 보안 등 개념부족
